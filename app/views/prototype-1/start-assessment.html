Cyber health framework self-assessment

This framework is based on a minimum cyber security threshold that all local authorities should meet. It has four overarching sections and 17 common categories of security controls and requirements that have been developed from core standards.

How to use this tool:

  • Click on the categories below to explore each one in more detail.
  • You will need to answer a series of questions in each category to determine if you are currently meeting the requirements for them.
  • Once you’ve finished a category, you’ll be shown if you met the requirements or not, and will be provided with guidance to help you improve.
